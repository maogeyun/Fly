# Impeccable — 飞机模式

## Design Context

### Users

- **核心场景**：纯朋友圈挑战——为社交传播与较劲而使用，短时、高频、偏娱乐。
- **使用情境**：多在安全环境下的即兴挑战（草地、床上方等），与现有安全文案一致。
- **要完成的任务**：抛机 → 看轨迹与评分 → 晒朋友圈或刷排行榜/历史，形成「好笑 + 想再比一次」的闭环。

### Brand Personality

- **三词气质**：戏谑、负责任、带火药味的轻松竞技。
- **语气**：可以调侃、有梗；安全与风险说明必须醒目、可信，不拿人身与设备安全开玩笑。
- **情绪目标**：刺激、好笑、竞技感——分数与排行是主要情绪引擎。

### Aesthetic Direction

- **大方向**：**黑白手绘 + 纸质质感**，唤起童年折纸飞机、草稿纸、铅笔/墨水线条的联想；在信息架构上可承接现有功能，但视觉从「暮光航线」式深色 UI **整体迁移到纸面语言**（而非在旧霓虹深色上小修小补）。
- **视觉隐喻**：笔记本纸纹、手画航线、纸飞机简笔动势；装饰与分区用手绘线框、略不规整的轮廓，避免默认「炫酷科技大屏」气质。
- **主题**：默认以 **浅色纸底 + 深色手绘线条/墨迹** 为主，最贴「童年纸飞机」与朋友圈截图习惯；若后续需要夜间场景，可用「深灰纸、卷边/压痕」做纸质暗色变体（非首轮刚需）。
- **参考 / 反参考**：无固定外部参考。**反参考**为强玻璃态、霓虹渐变、过度「控制台/仪表盘」 UI；与纸质感冲突时优先保纸面统一。

### Technical Constraints（微信小程序）

- 排版以系统字体栈为主（如 PingFang SC 等）；手绘感更多靠 **描边、纸质底纹（图或 pattern）、略手作的圆角与分割线**，而非堆叠非系统展示字体。
- 关键信息不依赖「只有动画才能看懂」；尊重用户对减弱动效的偏好（能关的过渡尽量可收敛）。

### Design Principles

1. **戏谑不越界**：版式再轻松，免责与风险提示的层级与可读性不可被「玩梗」压过。
2. **纸与笔优先**：层次与氛围用纸张纹理、手绘线框、墨迹轻重表达；少用强发光、厚重拟物阴影链。
3. **竞技感要一眼扫懂**：分数、排行、对比在涂鸦纸上仍保持清晰的信息层级（字号阶梯、留白分块、对齐节奏）。
4. **截图友好**：报告与成绩关键界面在发朋友圈截图时主体清晰、风格可辨识。
